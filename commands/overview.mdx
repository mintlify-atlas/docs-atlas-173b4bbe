## Command List

| Command | Description |
|---------|-------------|
| `/8ball` | Ask Jill for guidance with AI-powered predictions |
| `/surprise` | Let Jill pick a random drink for you |
| `/identity` | Pull forensic alias history for a user |
| `/mix` | Prepare beverages from real cocktails or VA-11 HALL-A specials |
| `/palette` | Extract an interactive color palette from an image |
| `/colors` | Pick a color from the public palette |

### Game Commands

| Command | Description |

| Command | Description |

## Permission Levels

Most commands are available to all users. Commands requiring elevated permissions are marked in their respective documentation pages:

- **Everyone**: No special permissions required
- **Manage Messages**: Moderate content and triggers
- **Manage Roles**: Role management commands
- **Manage Server**: Server configuration access
- **Administrator**: Full access to all features
- **Developer Only**: Reserved for bot owner

<Note>
Server administrators can customize command permissions using `/config permission`.
</Note>
